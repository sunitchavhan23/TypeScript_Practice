# TypeScript Setup and Execution Guide

## 1. Install Node.js

- TypeScript requires Node.js. Download and install Node.js from the official website.
- Check installation:
    **node -v**
    **npm -v**

## 2. Install TypeScript
- Install TypeScript globally using npm:
    **npm install -g typescript**

- Verify installation:
    **tsc -v**


## 3. Create a TypeScript File
- Create a file with the `.ts` extension.
    Example:
    **filename.ts**


## 4. Compile TypeScript File
- Use the TypeScript compiler to convert the `.ts` file into JavaScript.
    **tsc filename.ts**

- This command generates:
    **filename.js**

## 5. Run the Generated JavaScript File
- Use Node.js to execute the compiled JavaScript file.
    **node filename.js**


## Example Commands
1. npm install -g typescript
2. tsc filename.ts
3. node filename.js
